Authentification (application Android)

Le programme Authentification permet à un utilisateur
  - de s'inscrire 
  - de s'identifier avec son adresse email et son mot de passe
  
Les données de l'utilisateur sont stockées sur une base de données MySQL (superAuthentification.sql --> cf repository MySQL).
La communication entre la base de données et l'application se fait par l'intermédiaire de scripts php ( cf dossier Authentification dans le repository PHP)
